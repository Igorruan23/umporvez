# Umporvez - Seu Aplicativo Motivacional em Flutter

## Descrição
O Umporvez é um aplicativo desenvolvido em Flutter projetado para acompanhar e motivar você em direção a um objetivo específico. Defina a meta de dias para atingir o seu objetivo e mantenha-se motivado ao longo do caminho.

## Instalação
1. Clone este repositório: `git clone https://github.com/Igorruan23/umporvez.git`
2. Navegue até o diretório do projeto: `cd umporvez`
3. Instale as dependências: `flutter pub get`
4. Execute o aplicativo: `flutter run`
Certifique-se de ter o ambiente de desenvolvimento Flutter configurado corretamente antes de iniciar a instalação.

## Dependências
Este projeto utiliza as seguintes dependências:

- [sqflite](https://pub.dev/packages/sqflite) v2.3.0
- [google_fonts](https://pub.dev/packages/google_fonts) v6.1.0
Para instalar, inclua as dependências no seu arquivo pubspec.yaml e execute `flutter pub get`.
